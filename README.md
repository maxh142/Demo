# Demo

Demonstration of Typescript and JSON application using a product catalog program.

JSON array is used to store data and verified using a zod schema. Several TS concepts such as functions, maps, filters, enum, etc. used.

Steps to run: \
npm install \
npm run dev

Notes: \
JSON data usage is simple, but building it into a project like this and integrating it with other concepts (zod, package.json) helped my understanding greatly. \
Code has not been thoroughly tested.

***AI Usage*** \
AI was used to come up with the catalog idea and generated the catalog JSON data. Also used with path and folder formatting.
